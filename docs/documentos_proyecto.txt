Ejecución de scripts SQL:
========================
Para ejecutar los scripts en orden:
1. cd sql/
2. psql -U postgres -f 00_creacion_base_datos.sql
3. psql -U postgres -d academia_nexus -f 01_creacion_tablas.sql
4. psql -U postgres -d academia_nexus -f 02_triggers_reglas_negocio.sql
5. psql -U postgres -d academia_nexus -f 03_procedimientos_almacenados.sql
6. psql -U postgres -d academia_nexus -f 04_vistas_almacenadas.sql
7. psql -U postgres -d academia_nexus -f 05_vistas_joins_complejos.sql
8. psql -U postgres -d academia_nexus -f 06_conservas_utiles.sql
9. psql -U postgres -d academia_nexus -f 07_datos_prueba.sql
